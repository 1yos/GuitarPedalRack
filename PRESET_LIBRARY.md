## Preset Usage Guide

### How to Load Presets:

1. Open preset browser (top bar dropdown)
2. Filter by category or search by name
3. Click preset to load
4. Adjust to taste

### How to Save Custom Presets:

1. Adjust parameters to your liking
2. Click "Save Preset" button
3. Enter name, category, tags
4. Preset saved to user folder

### Preset File Location:

- **Windows:** `C:\Users\[username]\AppData\Roaming\GuitarPedalRack\Presets\`
- **Mac:** `~/Library/Application Support/GuitarPedalRack/Presets/`

### Sharing Presets:

- Presets saved as `.gpr` files (JSON format)
- Export: Right-click preset → Export
- Import: Drag .gpr file into preset browser
